# ⚡ Silicon-Agnostic Engine v3.0 (Universal Logic)
### Hardware-Aware Infrastructure Monitoring & Optimization Suite

A professional-grade **Systems Engineering** and **DevOps** dashboard built to bridge the gap between high-level software logic and low-level silicon telemetry. This project demonstrates real-time hardware interrogation, energy economics, and automated system optimization.



---

## 🚀 Key Engineering Features

### 1. **Silicon Telemetry (ECE Core)**
* **Dynamic Instruction Routing:** Adapts logic execution based on CPU load and thermal headroom.
* **Energy Economics:** Real-time estimation of power draw (Watts) and operational cost (INR/hr) based on ALU activity.
* **Hardware Interrogation:** Monitors CPU velocity, RAM volatility, and Power Infrastructure (AC vs. Battery).

### 2. **DevOps & SRE Tools**
* **Infrastructure Reaper:** Automated scanning for rogue processes consuming excessive resources (>20% CPU).
* **Network Latency Monitor:** ICMP-based infrastructure pinging to ensure external connectivity stability.
* **Volatile Memory Purge:** Manual trigger for Python's Garbage Collector to optimize the heap in real-time.

### 3. **The Stress Test (Universal Logic)**
* A specialized mathematical loop using **Floating Point Arithmetic** (Square Roots & Sines) to stress the CPU and measure the "Silicon Load Score."

---

## 🛠️ Technical Stack
* **Backend:** Python 3.x, Flask (Web Framework)
* **Hardware Layer:** `psutil` (System Sensors), `subprocess` (Shell Commands), `gc` (Memory Management)
* **Frontend:** HTML5, CSS3 (Glassmorphism UI), JavaScript (Asynchronous Polling)
